# CrossSell AI — 低保真原型 v0.1

## 原型文件

**HTML 交互式原型**：`deliverables/product-strategy/wireframes-crosssell-2026-07-16.html`

双击 `.html` 文件即可在浏览器中打开，无需任何依赖。

## 页面清单

| 页面 | 路由 | 说明 |
|------|------|------|
| 工作台首页 | Dashboard | 3x统计卡片 + CTA快速启动 + 最近任务列表 |
| 内容生成页 | Create | Step 1 表单 → Step 2 进度动画 → Step 3 结果Tab切换 |
| 投放策略页 | Strategy | 产品选择器 + 渠道推荐卡片 + 预算分配饼图 |
| 数据中心页 | Analytics | 日期选择 + 4x统计概览 + 平台分布饼图 + 任务表格 |

## 设计规范

| 属性 | 值 |
|------|-----|
| 主色调 | #6366f1 (light) / #818cf8 (dark) |
| 字体 | Inter (Google Fonts) |
| 卡片 | 白色背景、圆角12px、阴影、灰边框 |
| 侧栏 | 240px宽，4个导航项+Logo+主题切换 |
| 模式 | 亮色/暗色一键切换 |

## 交互标注

- 🟡 黄色虚线框 + 顶部标签 = 用户流程标注
- 🔴 红色输入框 + 红色错误文本 = 表单验证态（留空提交时激活）
- 🔵 蓝紫色步骤指示器 = 进度步骤条（active/done/waiting三种状态）
- ⚡ 点击"开始创建"CTA → 自动跳转内容生成页
- ⚡ 内容生成有完整的3步流程：表单 → 模拟进度动画 → 结果Tab切换
- ⚡ Tab支持：短视频预览 | 文案矩阵 | 投放建议
