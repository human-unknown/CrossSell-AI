## 1. 用户痛点全景：「五重断裂」模型

瑞思调研发现跨境卖家在社媒营销上存在五个相互放大的结构性痛点：

```
内容产能黑洞 → 支撑不了多平台多语言碎片化需求 → 出素材慢、数量少
                                      ↓
                            素材生命周期更短（算法不停消耗）
                                      ↓
                            跨文化转化鸿沟（内容无法共鸣）
                                      ↓
                            投放ROI黑洞（钱花了没效果）
                                      ↓
                            预算缩减或账户被封 → 回到起点，恶性循环
```

### 痛点一：内容产能黑洞
> **"不做视频没流量，做视频做不过来"**

- 3人团队80个SKU，仅视频需求320条，传统产能仅30条/月
- 单条视频制作3-6小时，外包¥200-500/条，月支出¥8K-25K
- 卖家原话："不是你想不想批量的问题，是不批量你根本喂不饱算法"
- 情绪：焦虑、无力感、被算法裹挟

### 痛点二：多平台多语言碎片化
> **"一个产品要出5种尺寸、4种语言、3种风格，搞完已经想转行了"**

- 一个产品需适配5平台×3-4市场 = 15-20个内容变体
- 翻译≠本土化：非本地化内容完播率38% vs 本地化61%
- 67%卖家仍用直译中文文案投放英语市场
- 情绪：分裂感、重复倦怠、永远做不完

### 痛点三：素材生命周期短 & 算法朝令夕改
> **"好不容易跑通了一条素材，3天后就衰退了"**

- TikTok带货视频生命周期仅3-5天
- 维持一个爆品需要15-20条不同角度素材/月
- 平台算法每季度至少1-2次重大变更
- 中国卖家因违规被封禁占比24%
- 情绪：朝不保夕、如履薄冰、疲于奔命

### 痛点四：跨文化转化鸿沟
> **"内容能跑出去，用户看了也点赞，就是不买"**

- 中国卖家 vs 本土品牌KOL ROI差距：1:2.8 vs 1:5.1
- 仅37%中国卖家在海外社媒实现正向ROI
- 转化率0.9%-1.2%，远低于行业均值1.8%
- 核心问题：有内容触达，无文化共鸣；有播放量，无购买决策触发
- 情绪：困惑、挫败、有力使不出

### 痛点五：投放ROI黑洞
> **"钱花出去了，回不来；账户还被封了"**

- 37%中国DTC品牌TikTok首月出现负向现金流
- 62%独立站卖家Facebook首月ROI<1.0
- 广告CPC年度涨幅23%，CPM达$12.5
- 中国卖家广告被封率41%，高出亚太均值15%
- 情绪：恐惧、愤怒、被平台"收割"的无力感

### 差异化判断

**"三合一"定位基本成立，但需精准定义边界：**

- ✅ 在 SMB 自助 SaaS 赛道，无人实现"内容生产+本土化+投流"完整闭环
- ✅ 卖家当前真实工作流是多工具拼接（ChatGPT + CapCut + ElevenLabs + Later + AdsPolar），整合需求迫切
- ⚠️ Navos 已做到三合一，但是 B2B 企业级服务（RAAS），非 SMB 自助 SaaS
- ⚠️ 易蛙智能正在做，Beta 阶段，目标用户高度重叠，需持续监控

### 最应关注的两个竞品

1. **易蛙智能**：定位"Vibe Marketing"人机协同，已有智能混剪+多市场适配，投流模块即将上线。与 CrossSell AI 目标用户高度重叠。**但目前仍处于Beta阶段，尚未大规模推广。**
2. **Navos（钛动科技）**：全球首个Martech Multi-Agent，能力最完整。但定位B2B企业级服务（8000+企业），定价模式是服务费。**短期不会SaaS化下沉，但长期是威胁。**

## 4. 产品策略建议

### 核心定位修正

原始定位："效率工具"（省时间省钱）
→ **修正为："效果引擎"**（提升ROI）

理由：数析数据显示素材质量权重高达30-40%，且好素材CPM低31%。"多出素材"的价值远不如"出好素材+提升ROI"的价值——后者有直接的可量化效果，利于定价和说服评委。

### MVP 功能优先级重排

| 优先级 | 功能 | 理由 |
|--------|------|------|
| P0 | URL/商品链接→多平台视频一键生成 | 产能黑洞的直接解法，竞品标配，必须先有 |
| P0 | AI脚本+多语言本土化配音+字幕 | 本土化3.8倍转化差的核心抓手，竞争壁垒 |
| P0 | 平台规格自动适配（TikTok/Reels/Facebook/Pinterest） | 多平台碎片化的直接解法 |
| P1 | 批量变体生成 + A/B测试框架 | 素材生命周期短的解法，"用量对抗不确定性" |
| P1 | 跨平台一键发布排程 | 分发效率提升，但非MVP生死功能 |
| P1 | 内容效果评分 + ROAS基础看板 | 数据闭环的起点 |
| P2 | ROAS预测模型 + 智能预算分配 | 需要数据积累，V1.5 补齐 |
| P2 | 文化适配引擎（俚语/禁忌/审美偏好） | 深度本土化的壁垒型功能，但MVP可先用基础多语言 |
| P2 | 竞品内容监控 | 增值功能，非MVP必要 |

### 产品叙事建议（比赛用）

1. **开场钩子**：一个3人团队80个SKU，需要320条视频但只能月产30条 → 引出"产能黑洞"
2. **痛点呈现**：五重断裂模型 → 层层递进的故事线
3. **方案展示**：CrossSell AI 的"三步走"——输入产品 → AI工厂自动生成多平台本土化内容 → 智能投流建议
4. **数据验证**：本土化3.8倍转化差 + 成本降至1/10 + 素材质量对ROI的杠杆效应
5. **差异化**：三合一的SMB SaaS空白定位
6. **Demo展示**：一个产品→3个平台→3种语言→3条不同风格内容的实际对比

## ⚠️ 待确认 / 假设 / Non-goals

### 待确认
- 真实卖家是否认同"五重断裂"的排序？（需要通过访谈验证）
- 卖家对"内容全链路自动化"的付费意愿和价格区间？
- AI生成视频在TikTok等平台的流量分发是否被限制？
- Model Router API的视频生成（wan2.7-t2v）质量标准能否满足"可用"要求？

### 关键假设
- 假设卖家愿意从"多工具拼接"迁移到"一站式平台"（需验证切换成本）
- 假设中小卖家（月GMV $5K-$500K）是最大痛点群体（大卖家有内部内容团队）
- 假设TikTok Shop生态是内容需求最大的市场（需北美/东南亚数据交叉验证）

### Non-goals（MVP 不做的）
- ❌ 不做全自动投放执行（V1只做策略建议，不做API对接下单）
- ❌ 不做KOL/红人发现和管理（与Creally定位错开）
- ❌ 不做独立站建站或商品Listing管理
- ❌ 不做邮件营销/客服自动化
- ❌ 不做数字人定制（依赖第三方模型如HeyGen）
- ❌ 不做14平台全覆盖（MVP聚焦 TikTok + Instagram + Facebook 三个平台）
