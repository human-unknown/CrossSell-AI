## 核心Pipeline实现细节

### Pipeline A：短视频生产流水线

```
产品信息输入
  │
  ▼
[1. 脚本生成] ──> 调用 qwen/qwen3.7-max
  │  Prompt: "你是一个TikTok短视频脚本专家，目标市场{market}，
  │          平台{platform}。为以下产品生成一个15-30秒的
  │          种草短视频脚本，包含镜头描述和台词。"
  │  输出: JSON { scenes: [{visual, audio, duration}], ... }
  │
  ▼
[2. 配音生成] ──> 调用 qwen/qwen3-tts-instruct-flash
  │  语言根据 target_market 自动选择（en/ja/de/...）
  │  输出: audio.mp3
  │
  ▼
[3. 素材准备] ──> 本地素材库 + AI图片生成
  │  调用 qwen/wan2.7-image-pro 生成产品场景图
  │  输出: 一组图片素材
  │
  ▼
[4. FFmpeg合成] ──> 本地执行
  │  图片序列 + 音频 + 字幕叠加 + 转场效果
  │  平台适配：TikTok(9:16), Reels(9:16), Shorts(9:16)
  │  输出: final_video.mp4
  │
  ▼
[5. 返回结果]
```

### Pipeline B：多平台文案矩阵

```
产品信息 + 目标平台列表
  │
  ▼
[并行调用] ──> 对每个(platform × market)发起独立API调用
  │  Prompt 示例（TikTok+美国市场）:
  │  "你是跨境电商社媒运营专家。为{product_name}撰写面向
  │   {market}市场{platform}平台的推广文案。
  │   要求：本地化语气、包含emoji和hashtag、{platform}的
  │   最佳字数范围、突出{selling_points}。"
  │
  ▼
[结果聚合] ──> 按平台分组返回
```

### Pipeline C：投流策略建议

```
产品信息 + 目标市场
  │
  ▼
[调用推理模型] ──> qwen/qwq-plus 或 deepseek-r1
  │  Prompt: "基于以下产品信息和目标市场，给出投放策略建议：
  │          推荐优先投放平台、建议日预算分配、
  │          预估ROAS范围、素材方向建议。"
  │
  ▼
[返回结构化策略]
```

| 推理 | `deepseek-r1` | 复杂策略分析 |

## 关键约束与注意事项

1. **API Key是赛方发放的**（初赛通过后发放25,000 Credits），踩点期间可以先用阿里云百炼的免费额度测试
2. **视频生成时间可能较长**——必须用异步任务模式，不能阻塞HTTP请求
3. **FFmpeg合成是本地的**——不依赖云端视频API，自己控制合成质量
4. **多语言配音质量是关键**——比赛Demo时如果配音听起来像机器人会很扣分，需要测试不同语言的效果
5. **Prompt Engineering 是核心竞争力**——同一个模型，Prompt写得好坏直接决定输出质量
